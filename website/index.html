What is LinBoard?

A lightweight clipboard manager that brings the Windows Win+V experience to every major Linux desktop.

⚡

Instant access

Press Super+V from any app. Your full clipboard history appears in a searchable popup — pick an item and it pastes automatically.

🔒

Private by design

History is stored locally in a JSON file on your machine. Nothing is sent to the cloud. No account, no tracking, no subscriptions.

🐧

Built for Linux

Native support for GNOME, KDE, XFCE, Cinnamon, and MATE on both X11 and Wayland — with CopyQ-style auto-paste built in.

Everything you need

Powerful clipboard management without the bloat.

Full clipboard history

Text, URLs, and images — all saved and searchable. Stores up to 200 items locally (configurable in config.json).

Fast search

Type to filter instantly. Find that link or snippet you copied hours ago in seconds.

Auto-paste

Click or press Enter — LinBoard pastes into your previous window. Built-in uinput on Wayland and X11 (xdotool optional fallback).

Pin important items

Press P to pin snippets you use often. Pinned items stay at the top.

Keyboard-first

Navigate with arrow keys, paste with Enter, pin with P, close with Esc. Never leave the keyboard.

System tray

Runs quietly in the background. Starts automatically on login after install.

How it works

From install to paste in under a minute.

  1. 1

    Install with one command

    Run the installer in your terminal. It detects your distro, installs dependencies, and sets up the Super+V shortcut automatically.

  2. 2

    Copy as usual

    LinBoard runs in the system tray and silently saves everything you copy — text, links, and images.

  3. 3

    Press Super+V

    Your clipboard history opens. Search, scroll, or use arrow keys to find the item you need.

  4. 4

    Click or press Enter

    LinBoard pastes the selected item into your previous window — just like Windows Win+V.

Install LinBoard

One command. Fully automatic. Works on most Linux distros.

Terminal
curl -fsSL https://raw.githubusercontent.com/Foisalislambd/LinBoard/main/scripts/install-release.sh | bash

  • Detects your distro and CPU (apt, dnf, pacman, zypper)
  • Installs tray support, xclip, and uinput for auto-paste
  • Configures Super+V and autostart on login
  • Adds ~/.local/bin to your PATH

After install, log out and back in once if prompted — this activates the input group for auto-paste.

Frequently asked questions

Is LinBoard free?

Yes. LinBoard is completely free and open source under the MIT license. No ads, no subscriptions, no account required.

Where is my clipboard data stored?

Locally on your machine at ~/.config/linboard/data/clips.json. Images are saved in ~/.config/linboard/data/images/. Nothing leaves your computer.

Does it work on Wayland?

Yes. LinBoard supports both X11 and Wayland on GNOME, KDE Plasma, XFCE, Cinnamon, and MATE.

Super+V doesn't work — what do I do?

Make sure LinBoard is running (linboard or linboard-start), then run linboard install-shortcut. Check Settings → Keyboard for the LinBoard entry. Run linboard doctor for a full diagnostic.

Paste isn't working after install

Run linboard setup-paste and follow the instructions. You may need to log out and log back in to activate the input group for uinput auto-paste.

No tray icon on GNOME?

Install the AppIndicator extension for GNOME to show system tray icons.

How do I uninstall?

Run pkill linboard, then remove ~/.local/bin/linboard and the autostart entry. See the full uninstall guide on GitHub.
